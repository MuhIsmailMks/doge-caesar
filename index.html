How To Buy

Install a wallet like Trust Wallet or MetaMask and create a new wallet. Store your seed phrase securely.

Buy BNB from an exchange (such as Binance or Tokocrypto), then send it to your wallet address on the BNB Smart Chain (BSC) network.

Go to pancakeswap.finance, click Connect Wallet,” then select Trust Wallet or MetaMask. Make sure the network is set to BNB Smart Chain.

Use PancakeSwap to exchange BNB for $DOGE (BEP-20 version). Make sure the slippage is correct and click "Swap".
